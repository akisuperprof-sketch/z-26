# L/P/A プラン設計仕様書 (v0.1.0)

## 概要
舌診アプリをターゲットユーザーに合わせて3つの階層に分割し、それぞれのニーズに最適化された解析を提供する。

## 3プラン構成

### 1. Lite (簡易舌診)
- **判定軸**: 寒熱スペクトラム（横軸のみ）
- **項目**: 舌色、苔色、ヒアリング6問
- **目的**: 一般層への入り口。

### 2. Pro ( 본격判定)
- **判定軸**: 寒熱 × 虚実（4象限マトリクス）
- **項目**: 舌体形状、歯痕、苔量、ヒアリング15問
- **目的**: 意識高い層、鍼灸師助手レベル。

### 3. Academic (研究・専門)
- **判定軸**: 五行 × 寒熱虚実 (立体構造)
- **項目**: 舌診アトラス照合、五行分類、詳細ヒアリング25問
- **目的**: 専門家、臨床研究、教育用。

## 実装戦略
- `localStorage.getItem('DEV_FEATURES')` が `'true'` の場合のみ、新解析エンジン (`tongueAnalyzerRouter`) が有効化される。
- 各解析モデルは `services/analyzers/` 配下で独立して管理される。
